# List
> 리스트명 = [요소1,요소2,요소3]	
	
## 리스트의 슬라이싱	
```python
>>> a= [1,2,3,4,5]
>>> a[0:2]
[1,2]
```
	
## 리스트 연산
```python
>>> a = [1,2,3]
>>> b = [4,5,6]
>>> a + b
[1,2,3,4,5,6]
>>> a*3
[1,2,3,1,2,3]z
```
	
## 리스트 수정 삭제
```python
>>> a = [1,2,3]
>>> del a[1]
>>> a
[1,3]
>>> a.append(4)
[1,3,4]
>>>a.append([5,6])
[1,3,4,5,6]
```
	
## 기타 함수
```python
>>> a = [1,4,3,2]
>>> a.sort()
[1,2,3,4]
>>> a = ['a','b','c']
>>> a.reverse()
['c','b','a']
>>> a.index('a')
2
>>> a.pop()
a
>>> a
['c','b']
>>> a.insert('d','e')
>>> a
['c','b','d','e']
>>> a.remove('e')
['c',b','d']
>>> a.pop(1)
>>> a
['c','d']
>>> b = [1,2,3,4]
>>> c = [5,6]
>>> b.extend(c)
>>> b
[1,2,3,4,5,6]
```
